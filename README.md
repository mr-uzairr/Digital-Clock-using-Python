# Digital Clock using Python

This is a simple digital clock created using Python's Tkinter library. It shows the current time and updates every second.

## Requirements

- Python 3.x

Tkinter comes pre-installed with Python, so no need to install any external packages.

## How to Run

1. Download or clone the repository.
2. Run the following command in your terminal:

```bash
python digital_clock.py
```

A window will open showing the current time.
